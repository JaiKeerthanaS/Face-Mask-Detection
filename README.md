# Face Mask Detection using CNN

## Overview

This project detects whether a person is wearing a face mask in real time using a webcam.

## Technologies Used

- Python
- OpenCV
- TensorFlow
- CNN

 # Dataset used:
Face Mask Dataset from Kaggle.
Download and place it inside:

dataset/
├── with_mask/
└── without_mask/

## Features

- Real-time face detection
- Face mask classification
- Webcam integration

## How to Run

Install dependencies:

pip install -r requirements.txt

Train model:

python train_mask_detector.py

Run detection:

python detect_mask.py
